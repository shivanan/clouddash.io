Frequently Asked Questions

Everything you need to know about Cloud Dash

When will Cloud Dash be released?

We're currently in private beta. Public beta will start in Q3 2025, with a full release planned for Q4 2025.

How much will Cloud Dash cost?

Cloud Dash will be offered with a 14-day free trial, followed by a $30/year subscription. Beta testers will receive 3 months free after launch.

What macOS version is required?

Cloud Dash requires macOS 12 (Monterey) or newer to ensure the best performance and compatibility with the latest macOS features.

Is my AWS data secure?

Absolutely. Cloud Dash uses your existing AWS credentials and does not store any sensitive data on our servers. All communication is encrypted using industry-standard protocols.

Can I manage multiple AWS accounts?

Yes! Cloud Dash is designed to handle multiple AWS accounts with ease, allowing you to switch between them quickly.
